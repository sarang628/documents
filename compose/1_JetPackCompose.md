# [Jetpack Compose](https://developer.android.com/compose)
- native UI 빌드를 위한 안드로이드 권장 modern toolkit
- UI 개발을 '단순'하고 '빠르게' 해준다.

# [Why adopt Compose](https://developer.android.com/develop/ui/compose/why-adopt)
- [적은 코드](https://developer.android.com/develop/ui/compose/why-adopt#less-code)
  - 코틀린을 사용
  - 기존 xml 구현 보다 작성하기 간단함
- [직관적](https://developer.android.com/develop/ui/compose/why-adopt#intuitive)
  - 선언형 API 사용
  - 작은 단위 **stateless 컴포넌트** 생성 가능, 엑티비티, 프레그먼트와 분리되어 **재사용**, **test** 용이
- [개발 가속화](https://developer.android.com/develop/ui/compose/why-adopt#accelerate-development)
  - 기존 뷰와 Compose 상호 구현 가능
  - 안드로이드 Studio의 Preview 도구로 빠르게 UI 확인 가능
- [강력함](https://developer.android.com/develop/ui/compose/why-adopt#powerful)
  - Material Design, Dark theme, animations과 같은 안드로이드 플랫폼 API와 함께 사용 가능
  - 하지만 Compose UI API들은 위 플랫폼 API와 분리되어있어 Custom 하기도 매우 편리
