# [Gradle 이란?](https://docs.gradle.org/current/userguide/userguide.html)

- **자동화 빌드 도구**.
- **신뢰성있는 적응형** 자동화 빌드 도구.
- **우아하고 광범위한 선언형 언어를 포함한** 신뢰성있는 적응형 자동화 빌드 도구이다.


# [Plugin 이란?](https://docs.gradle.org/current/userguide/plugins.html)

- 플러그인은 컴파일 코드, 테스트, 아티팩트 배포 등을 위한 **캡슐화된 로직**.
- 플러그인 소스는 3가지로 분류
  - 코어 - 그래들 내 포함
  - 커뮤니티 - 그래들 플러그인 포털에 있음
  - 로컬 - 개발자가 직접 개발
