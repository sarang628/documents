# [OverView](https://coil-kt.github.io/coil/)
- 안드로이드와 컴포즈 멀티플랫폼을위한 이미지 loading library
- 빠름 : 메모리와 디스크 캐싱, 이미지 다운 샘플링, 자동 요청 보류, 취소 등 성능 최적화
- 가벼움 : 코틀린, 코루틴, Okio, Google's R8 code 슈링커만 의존성 추가
- 쉽게 사용 : 코틀린의 특징을 활용하여 간단하고, 보일러 플레이트 최소화
- 모던 : kotlin-first 채용과 Compose, Coroutines, Okio, OkHttp, and Ktor 같은 모던 라이브러리와 상호호환 가능
- Coil : **Co**routine **I**mage **L**oader.
