# [안드로이드의 코틀린 우선 전략 (Kotlin-first)](https://developer.android.com/kotlin/first)

2019년도부터 구글에서 도입 시작

**표현력이 뛰어나고 간결한 언어**

**일반적인 에러를 줄여주고 기존 코드와 쉽게 통합이 가능한** 표현력이 뛰어나고 간결한 언어

구글은 안드로이드 개발에 코틀린을 사용하기 위해 코틀린 설립재단에 코파운더로 활동

컴파일 성능과 빌드 속도에 지속적으로 투자중

## 코틀린 우선 전략을 하게된 이유?

- 표현력이 뛰어나고 간결함
- 안전한 코드 구현 가능
- 자바와 상호 운용 가능
- 구조화된 동시성 지원
    - blocking 코드 만큼 비동기 코드를 쉽게 만들게 해줌
    - 네트워크 호출에서 로컬 데이터 접근까지 백그라운드 작업을 극적으로 단순화 해줌

## [Kotlin-first 의 의미?](https://developer.android.com/kotlin/first#what)
새롭개 개발하는 안드로이드의 개발 툴과 내용들 Jetpack libraries, sample 문서, training content 등이
코틀을 기반으로 만듬.

## [구글의 앱들도 코틀린 사용](https://developer.android.com/kotlin/first?utm_source=android-studio-app&utm_medium=app#use)
- 구글의 70개 이상의 앱을 코틀린으로 제작.
- 구글 홈 팀에선 33% 이상의 코드베이스 감소와 30% NPE 감소를 경험.
